⚙ Configuración
Estado de conexión
Conectar base de datos (solo una vez)
Sigue estos pasos para que tus datos se guarden en la nube y puedas verlos en cualquier dispositivo.
1
Ve a console.firebase.google.com e inicia sesión con tu cuenta de Google.
2
Clic en "Agregar proyecto" → escribe un nombre (ej: misfinanzas) → desactiva Analytics → "Crear proyecto".
3
En el panel del proyecto, clic en el ícono </> (Web) → nombre de la app: web → "Registrar app" → copia el objeto firebaseConfig que aparece.
4
En el menú izquierdo: Compilación → Firestore Database → Crear base de datos → Modo producción → Elegir región: us-east1 → Listo.
5
En Firestore, clic en "Reglas" y reemplaza todo con este texto:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }

Clic en "Publicar".
6
Abre el archivo firebase-config.js con el Bloc de notas y pega ahí los valores que copiaste en el paso 3. Guarda el archivo y recarga esta página.
Mientras tanto, todos tus datos se guardan localmente en este dispositivo. Al conectar Firebase, se migrarán automáticamente a la nube.
